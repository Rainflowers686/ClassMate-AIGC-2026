# Stage 7 Live Demo Questions And Inputs

## 1. 可口播给 ASR 的短课堂内容

### 高数：数项级数

同学们，今天我们复习数项级数的收敛判别。判断级数收敛前，先看通项是否趋于零；如果通项不趋于零，级数一定发散。几何级数的关键是公比绝对值小于一时收敛。p 级数在 p 大于一时收敛，在 p 小于等于一时发散。比较判别法适合处理正项级数，比值判别法和根值判别法常用于含阶乘或 n 次方的项。交错级数还要看绝对收敛和条件收敛的区别。

### 物理：电磁感应

今天讲电磁感应。磁通量描述磁场穿过某个面积的多少，法拉第电磁感应定律说明感应电动势等于磁通量变化率的负值。这个负号来自楞次定律，表示感应电流的方向总是阻碍磁通量的变化。导体棒在磁场中运动会产生动生电动势，本质是洛伦兹力推动自由电荷移动。自感和互感都体现了电流变化引起磁通变化，最后都涉及能量转换。

### 离散数学：关系与偏序

这节课讲二元关系和偏序。集合 A 上的二元关系可以看成 A 乘 A 的子集。自反表示每个元素都和自己有关系，对称表示如果 a 关系 b，那么 b 也关系 a，传递表示 a 关系 b 且 b 关系 c 时 a 关系 c。等价关系要求自反、对称、传递。偏序关系要求自反、反对称、传递。哈斯图用于表示偏序结构，极大元不一定唯一，最大元如果存在则唯一。

### C++：指针与引用

今天复习 C++ 指针和引用。指针保存变量地址，可以为空，也可以重新指向别的对象。解引用指针时要确保它不是空指针，否则会产生未定义行为。引用必须在定义时初始化，并且初始化后不能改绑到另一个对象。函数参数传递中，值传递会拷贝，指针传递可以修改外部对象，引用传递语法更接近普通变量。const 引用常用于避免拷贝并防止函数修改实参。

### 马原：实践与认识

今天讲实践与认识。实践是认识的来源，也是认识发展的动力。人们在实践中提出问题，在改造世界的过程中形成感性认识，再上升到理性认识。真理不是主观感觉决定的，检验真理的标准是社会实践。认识具有反复性和无限性，因为客观世界复杂，人的认识条件也会变化。学习这个部分时，要把实践、认识、真理和检验标准联系起来。
