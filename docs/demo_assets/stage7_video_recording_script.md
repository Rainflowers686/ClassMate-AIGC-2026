# Stage 7 Video Recording Script

录屏通用注意事项：

- 录屏前开启勿扰模式。
- 不录密钥导入过程。
- 不拍本地配置文件、完整日志、内部模型输入/消息/推理字段。
- 不拍私人账号、真实学生隐私、云真机后台。
- Live ASR 只说“实验模式”，不说真实 vivo ASR provider 已完成。
