# Real Device Test Manual - 1.14.4 / 117

## A. ASR 与录音

1. 打开资料导入页的课堂录音卡片。
2. 若设备没有系统语音识别服务，预期看到“打开语音设置 / 仅录音 / 粘贴转写文本”。
3. 点击“仅录音”，预期录音保存，不生成假 transcript。
4. 点击“粘贴转写文本”，预期进入转写编辑入口，确认后生成学习闭环。
5. 若系统 ASR 可用，点击“开始录音并实时转写”，预期 partial 只临时显示，final 去重后进入转写段落。

## B. 蓝心失败与重试

1. 在未配置或断网环境运行课程分析。
2. 预期普通页面显示友好原因：未配置、网络不可达、超时、鉴权失败或返回为空。
3. 预期本地整理结果不被清空，重试按钮可用。
4. 开发者诊断可显示技术码，但普通页面不得显示真实 AppKey、Authorization 或完整 endpoint。

## C. 微测题相关性与详解

1. 用一段明确学科材料生成课程。
2. 打开微测，检查每题是否围绕知识点。
3. 答题后检查：正确答案、为什么正确、其他选项为什么不对、对应知识点、证据摘录均可见。
4. 若材料不足，预期显示“资料不足，暂不能生成可靠微测”，不硬造题。

## D. 反馈即时优化

1. 对一道题点击反馈，选择“题目不准确”。
2. 预期当前题被移除或标记不再推荐。
3. 预期立即出现同知识点替换题，且替换题有答案详解和证据。
4. 对知识点反馈“不准确”，预期该知识点显示需复核并尝试重写摘要。
5. 对证据反馈“不相关”，预期证据降级为待核对或替换为更相关证据。

## E. 复习计划与课程相关知识点

1. 打开复习页。
2. 预期只看到本课核心知识点、薄弱知识点、待复核点、推荐微测和今日复习任务。
3. 预期不出现 `L3`、provider trace、raw id、semantic index、技术流水。
4. 打开知识页/课程页的相关知识点，预期来源是本课 evidence / OCR / transcript / material text，不声称外部 API 总结。

## F. 记录失败材料

- 截图：ASR 卡片、蓝心失败卡片、微测详解、反馈替换题、复习页。
- 日志：只保存脱敏诊断，不记录 AppKey、Authorization、Bearer 或完整用户隐私原文。
