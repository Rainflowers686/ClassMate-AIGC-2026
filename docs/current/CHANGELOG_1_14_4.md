# ClassMate 1.14.4 / versionCode 117

提交主题：真机 ASR、蓝心失败、微测相关性与反馈即时优化回归修复。

## 修复内容

- ASR：无系统语音识别服务时不再只报错；用户可以打开语音设置、仅录音、粘贴转写文本。
- ASR 准确率：系统识别请求按界面语言传入 `zh-CN` / `en-US`，并去重重复 final transcript。
- 蓝心：失败原因按未配置、鉴权失败、网络不可达、超时、返回为空等类别展示，普通页面不泄露 raw code 或密钥。
- 微测：题目必须绑定知识点和证据；缺绑定的模型题被过滤；本地兜底题只使用本课材料。
- 答案详解：正确项、错误项、知识点、证据摘录一起展示。
- 反馈：题目反馈生成同知识点替换题；知识点/证据反馈进入需复核和重写路径。
- 复习计划：普通页面只展示学科知识点、薄弱点、待复核点、相关知识点和证据，不展示技术流水。

## 风险

- 官方 ASR/TTS/OCR 的真实网络成功仍取决于 AppKey、权限、设备 ROM 和接口状态。
- 系统语音设置 Intent 在不同 ROM 上可达性不同，ClassMate 已逐级 fallback 到应用详情和系统设置。
- 蓝心失败时仍保留本地整理版，不把本地 fallback 冒充蓝心结果。

## 验证重点

1. 无系统语音识别服务的手机：录音仍能保存，且可进入手动转写。
2. 蓝心未配置或网络失败：页面显示友好原因和重试，不清空本地结果。
3. 微测题：题干、选项、答案详解都能回到知识点和证据。
4. 反馈题目问题：原题被移除，新题立即可见且带详解。
5. 复习计划：不出现 `L3`、raw id、provider trace 等技术词。
