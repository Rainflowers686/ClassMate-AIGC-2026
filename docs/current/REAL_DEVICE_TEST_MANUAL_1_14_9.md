# ClassMate 1.14.9 真机复测手册

## A. 练习完成不自动退出

操作：
1. 打开已有课程。
2. 进入知识点时间线，点击开始微测。
3. 完成所有题目并点击最后的“完成练习”。

预期：
- 仍停留在练习页。
- 显示完成摘要、正确/错误情况和涉及知识点。
- 只有点击“返回课程”或系统返回时才离开。

## B. 复习计划过滤课堂强调词

操作：
1. 导入或粘贴文本：`同学们注意，重点来了，这个地方可能考。下面看牛顿第二定律，物体的加速度与所受合外力成正比，与质量成反比，公式 F=ma。大家记一下，作业截图上传。`
2. 生成课程。
3. 打开复习计划和课程总结。

预期：
- 知识点应围绕牛顿第二定律、加速度、合外力、质量或 F=ma。
- “同学们注意 / 重点来了 / 大家记一下 / 作业截图上传”不能作为知识点、相关知识点或微测题主题。

## C. 微测入口一致

操作：
1. 在课程详情点击微测统计。
2. 回到课程后从知识点时间线点击开始微测。
3. 从复习计划进入本课知识点复习。

预期：
- 三个入口都进入 Practice 主流程。
- 如果证据页有题，知识点时间线也能拿到同源题目。
- 无题时显示重新生成微测和补充资料入口，不展示空白页。

## D. BlueLM 三档模式

操作：
1. 在设置中保持 BlueLM 配置。
2. 分别选择快速、均衡、专业模式触发课程分析或 AI 精修导出。
3. 可运行 `powershell -ExecutionPolicy Bypass -File scripts\qa\provider_live_smoke.ps1` 做脱敏 smoke。

预期：
- 用户页面只显示“蓝心 / 蓝心大模型”，不显示底层模型名。
- 快速/均衡不启用深度思考，专业启用深度思考。
- 专业模式允许数分钟等待，提供取消；失败后保留本地整理版。
- dry-run 或 smoke 无配置时输出 SKIP，不算真机失败。

## E. 安全检查

导出学习包后检查：
- 无 AppKey、Authorization、Bearer 值。
- 无 raw id、provider trace、reasoning_content。
- fallback 结果标注为本地整理版，不冒充 BlueLM。
