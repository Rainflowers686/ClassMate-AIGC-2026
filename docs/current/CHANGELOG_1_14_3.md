# Changelog - ClassMate 1.14.3 / versionCode 116

本轮定位：真机录音/转写 readiness、反馈即时学习优化、复习知识摘要和题目详解增强。

## 1.14.3 / 116

### 修复与增强

- 系统语音识别不可用时，录音卡片提供“打开语音设置”入口，并继续保留录音保存、手动转写、字幕导入 fallback。
- 题目、证据、知识点、复习计划反馈会立即作用到学习闭环：生成替换题、重写知识点摘要或重新组织复习重点。
- 复习页从技术统计切换为“本课知识点复习”，展示核心知识点、相关知识点、证据摘录和需复核提示。
- 知识页增加当前课程内相关知识点摘要，不使用外部搜索 API，不混入其他课程材料。
- 替换题包含答案详解、选项解释和证据摘录，减少“只显示答案”的问题。

### 风险与边界

- 官方 OCR/ASR/TTS 网络能力仍需真实 AppKey、设备权限、接口权限和网络环境验证。
- “打开语音设置”是否直达目标页面取决于 ROM；失败时用户仍可手动进入系统设置，录音和手动转写不受影响。
- 相关知识点摘要只基于当前课程 snapshot 内的知识点和 evidence，不代表外部百科或在线搜索结果。

### 建议复测

1. 在无系统语音识别服务设备上开始录音，确认显示语音设置入口，停止后录音仍保存。
2. 对一道题反馈“题目不准确”，确认出现替换题并进入微测候选。
3. 对知识点反馈“讲解不清楚”，确认摘要更新且复习计划包含该知识点。
4. 打开复习页，确认看到本课核心知识点与相关知识点，而不是 L3 技术统计。
