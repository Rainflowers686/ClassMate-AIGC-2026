# AI Study Pack Product Spec

## 目标

把当前课程闭环产物整理成可打印、可分享、可交付的学习资料，而不是简单复制原文。

## 输入

- L3 snapshot。
- KnowledgePoint。
- Evidence。
- Quiz / Practice result。
- WrongBook / weakness。
- ReviewPlan。
- Material source summary。

## 输出

- 普通 Study Pack。
- AI 精修版 Study Pack。
- 课程精华音频脚本文本。
- PDF / HTML / DOCX / Markdown / Text artifact。

## 用户体验边界

- 普通导出不等待蓝心。
- AI 精修需要用户主动触发。
- 精修运行中显示阶段和慢响应提示。
- 失败时说明原因和下一步，不覆盖普通版。

## 不做

- 不生成声音复刻。
- 不把系统 TTS/脚本包装成官方 TTS 成功。
- 不把缺证据内容写成确定事实。
