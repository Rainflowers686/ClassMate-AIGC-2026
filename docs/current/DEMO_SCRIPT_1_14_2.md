# Demo Script - ClassMate 1.14.2

## 3 分钟版

1. 资料篮添加图片和一段课堂文本。
   - 预期：图片/OCR 草稿与文本资料可见。
2. 生成知识结构。
   - 预期：出现知识点、重点解释、证据。
3. 打开证据详情。
   - 预期：能回到文字或图片来源。
4. 做图片微测并故意答错。
   - 预期：显示解析、正确答案、证据入口。
5. 标记反馈。
   - 预期：出现需复核/薄弱点/复习任务。
6. AI 精修导出。
   - 预期：精修版可生成或 fallback，普通导出仍可用。
7. 简短说明官方能力。
   - 话术：官方 ASR/TTS/OCR 已有代码路径和 fallback，真实网络成功需 AppKey 与真机验证。

## 5 分钟版

在 3 分钟版基础上增加：

1. 录音/ASR fallback。
   - 系统 ASR 可用时展示实时转写；不可用时展示录音保存和手动转写。
2. TTS 听背。
   - 官方 TTS 待验证时使用系统 TTS 或文稿。
3. 心流背景音乐。
   - 展示授权循环背景音。
4. 外部搜索浏览器入口。
   - 强调这是浏览器搜索，不是 API 推荐。

## 8 分钟版

在 5 分钟版基础上增加：

1. 官方能力矩阵。
2. 云端/端侧/本地 fallback 架构。
3. SafeExportText、安全与隐私。
4. 测试证据：unit tests、current_preflight、cloud_device_precheck。
