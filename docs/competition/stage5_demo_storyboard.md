# Stage 5 复赛展示分镜

目标时长：8-10 分钟。重点展示 ClassMate 不是普通录音总结工具，而是“多来源课堂资料 -> 可验证知识点 -> 证据约束问答 -> 微测 -> 自适应复习 -> 可分享报告”的学习闭环。

| 时间 | 镜头 | 页面/操作 | 讲解重点 | 风险提醒 |
|---|---|---|---|---|
| 0:00-0:50 | 项目定位 | 首页 Dashboard | ClassMate 面向课堂学习，不只做记录和总结。核心闭环是理解、证据、测验、复习和反馈。 | 不要夸大未实现能力。 |
| 0:50-1:40 | 官方主路径 | 设置模型配置和 BlueLM 诊断 | 官方 BlueLM/qwen 是参赛合规主路径；Compatible 只是展示增强；LocalFallback 保证系统可用。 | 不录到调试导入明文。 |
| 1:40-2:45 | 多来源导入 | 导入 flow：文本、课件 OCR、字幕/转写稿 | 追平录音总结类产品的输入能力入口，同时保持“不上传、不爬取、不伪装真实 ASR/OCR”。 | 文案必须显示诚实边界。 |
| 2:45-3:25 | 统一资料包 | 资料篮和分析设置 | 文本、OCR、字幕都进入统一 MaterialBundle，再由同一 CourseAnalyzer 处理。 | 不展示模型输入原文构造细节。 |
| 3:25-4:30 | Timeline + Evidence | 课程详情进入知识时间线和证据链 | 每个知识点不是纯总结，而是能回到课堂原文或 OCR 来源。 | 证据 quote 不要包含隐私。 |
| 4:30-5:30 | Ask This Lesson | 提问 grounded / partial / not_found | Ask 不是普通聊天，必须受本节课证据约束。无依据时拒绝胡编。 | 不问敏感或外部无关问题。 |
| 5:30-6:30 | Quiz + ReviewTask | 微测答题，进入复习 tab | 题目有证据解释；答错、太难、需要例题等反馈进入 LearningStore 和复习任务。 | 不说复习算法已覆盖所有学习科学模型。 |
| 6:30-7:35 | Course Library | 课程库进入 Course Detail | 课程按名称聚合，课程详情提供 Timeline、Ask、Quiz、Review、Sources、Export 子入口。 | 不把 History 说成云同步。 |
| 7:35-8:35 | Export Center | 导出 PDF / Markdown / Word 兼容 HTML / Slides HTML，演示分享 | 用户不需要去 app 私有目录找文件，可保存到选择位置或系统分享。 | 说明 Word/Slides 是兼容 HTML，不是真原生格式。 |
| 8:35-9:20 | 差异化 | 对比录音总结类产品 | ClassMate 追平基础输入，但超越点是证据链、微测、自适应复习、课程库和学习状态。 | 不贬低竞品，不抄袭竞品。 |
| 9:20-10:00 | 诚实边界 | 能力路线图 | 不做平台爬取；真实 ASR/OCR、声纹身份识别和底噪增强仍是后续接入或依赖 provider 能力。 | 不声称未实现能力已完成。 |

## 20 秒收尾

ClassMate 的目标不是把课堂内容变成一段摘要，而是把课堂资料变成可以追溯、可以练习、可以复习、可以导出的学习闭环。当前版本已经跑通官方 BlueLM 文本理解主路径，并把多来源资料、证据链、微测、复习任务和导出中心连接起来。后续接入 vivo ASR/OCR 后，这套结构可以自然扩展到真实录音、课件和板书。
