## 评审解释

- “已实现”表示可在复赛版直接演示。
- “实验模式”表示有入口和边界说明，但依赖设备或系统能力。
- “开发中/待验证”表示当前正在由 Stage 7C 完成，不应在答辩中说成稳定完成。
- “暂缓”能力不会作为复赛承诺。
