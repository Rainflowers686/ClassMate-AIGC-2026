# Stage 7 Final Demo Script

适用场景：8-10 分钟复赛现场演示或录屏配音。默认只展示脱敏界面；不要展示本地配置文件、密钥输入框、完整鉴权头、模型原始请求/响应、模型内部推理内容。

## 1. 开场 30 秒：ClassMate 定位

讲解词：
ClassMate 不是普通录音总结工具。它面向课堂学习，把“课堂资料”变成可验证的知识点、原文证据链、微测、自适应复习、需要多练的练习入口和可导出的学习报告。我们做的是学习闭环，而不是只把一段录音变成摘要。

操作步骤：
- 启动 App，停留在首页 Dashboard。
- 指向今日状态、导入资料、Live Companion、最近课程和模型状态 chip。

画面重点：
- 首页不是长说明页，而是学习任务入口。
- 看到“导入课堂资料”和“Live Companion”两个主入口。

不要展示：
- Debug 导入密钥页面。
- 本地配置文件内容。

## 2. 官方合规路径 60 秒

讲解词：
参赛合规主路径是 Official BlueLM/qwen3.5-plus。Compatible Demo 只用于展示兼容增强，LocalFallback 用来保证网络不可用时系统仍可演示基本学习流程。密钥只通过本机 debug 配置进入运行态，不进入仓库、日志、历史、导出或截图。

操作步骤：
- 进入 Settings。
- 展示模型配置卡：Official BlueLM、Compatible Demo、LocalFallback。
- 运行 BlueLM connection test，如已有成功结果可展示 OK 状态。

画面重点：
- Official BlueLM 是主路径。
- LocalFallback 是兜底，不替代官方路径。
- 日志只显示 provider/status/latency 等短字段。

不要展示：
- 密钥明文。
- 鉴权头明文。
- 完整模型输入输出。

## 3. 多源导入 90 秒

讲解词：
ClassMate 已经不只是粘贴文本。复赛版支持文本、Markdown、手动 OCR 资料流、SRT/VTT/TXT 字幕或转写稿、Live 手动课堂，以及 Live ASR 实验模式。Live ASR 依赖设备系统 SpeechRecognizer，不保存原始音频，不后台录音；vivo ASR/OCR provider 是决赛接入方向。

操作步骤：
- 进入导入课堂资料。
- 展示资料类型：粘贴文本、示例课堂、Markdown、OCR/PPT/板书、字幕/转写稿。
- 加入一段 OCR 或 SRT/VTT 测试资料到资料篮。
- 展示 Live 页面中的手动/实验 ASR 诚实提示。

画面重点：
- Material Tray 将不同来源统一成课堂资料。
- OCR/ASR 未完成的 provider 不被包装成已完成能力。

不要展示：
- 真实学生隐私材料。
- 本地敏感路径。
- 音频原始文件内容。

## 4. Knowledge Timeline 90 秒

讲解词：
BlueLM 不是直接输出一个不可追溯的总结。ClassMate 会把课堂资料整理成知识时间线，每个知识点必须带证据。证据来自课堂文本、字幕、手动 OCR、Live 片段等来源 marker。校验器负责防止没有依据的知识点进入最终结果。

操作步骤：
- 点击生成知识时间线。
- 打开 Timeline。
- 点开一个知识点和证据详情。

画面重点：
- 知识点标题、重要性、难度。
- Evidence quote 和来源 marker。
- 证据可追溯到本节课资料。

不要展示：
- 模型原始响应体。
- 调试日志全文。

## 5. Ask This Lesson 90 秒

讲解词：
“问这节课”不是开放式闲聊。系统只基于本节课资料和已提取证据回答，回答会标注 grounded、partial 或 not_found。若本节课没有依据，系统应该拒绝胡编，并提示资料中没有明确证据。

操作步骤：
- 在课程详情或时间线进入 Ask This Lesson。
- 问一个直接有证据的问题。
- 问一个只有部分依据的问题。
- 问一个本节课没有依据的问题。

画面重点：
- answer。
- evidence refs 或 related knowledge points。
- groundedness 状态。

不要展示：
- 内部构造的完整模型输入。
- 供应商响应全文。

## 6. Quiz / Review / 需要多练 90 秒

讲解词：
ClassMate 会把知识点转成微测。答题后，正确、错误、太难、证据不对、需要多练等反馈会进入学习状态，推动复习任务和练习入口。需要多练不是一个标签，而是可以触发外部搜索词和练习方向，但 ClassMate 不抓取平台内容。

操作步骤：
- 进入 Quiz，完成一题。
- 展示答案解释和证据。
- 进入 Review，展示任务、优先级和薄弱点。
- 展示“需要多练”或练习搜索入口。

画面重点：
- 微测有解释和证据。
- Review 任务可解释。
- 练习搜索只是打开外部搜索结果。

不要展示：
- 任何平台抓取或下载行为。
- 私人账号页面。

## 7. Export Center 60 秒

讲解词：
导出中心解决云真机和普通用户找不到文件的问题。报告可以保存到用户选择的位置、下载目录兜底，或通过系统分享。支持 PDF、HTML、Markdown、TXT、MindMap、Word 兼容 HTML 和 Slides HTML。Word/Slides 是兼容 HTML，不冒充原生 docx 或 pptx。

操作步骤：
- 打开课程详情或 Export Center。
- 选择 PDF 或 Markdown。
- 保存到文件或系统分享。
- 展示报告里的课程概要、知识点、证据链、微测、复习计划和隐私说明。

画面重点：
- 文件可被用户拿到。
- 报告内容结构化、可打印。
- 导出不含敏感配置、内部请求或模型内部内容。

不要展示：
- 本地私有配置路径。
- Debug 原始日志。

## 8. 对比录音转写类产品 60 秒

讲解词：
录音转写类产品强在实时记录、转写和摘要。ClassMate 正在补齐这些输入能力，但差异化不是“再做一个录音工具”，而是围绕学习闭环：证据链、微测、自适应复习、需要多练、课程库和报告导出。

操作步骤：
- 展示多源导入。
- 展示 Timeline/Evidence。
- 展示 Review/Export。

画面重点：
- 输入能力追平基础。
- 学习闭环形成优势。

不要展示：
- 对竞品贬低性描述。
- 夸大尚未完成的 provider 能力。

## 9. 诚实边界 30 秒

讲解词：
复赛版不做第三方平台视频爬取，不伪装 vivo ASR/OCR provider 已接入，不做声纹身份识别，不自研底噪处理，云同步和团队协作暂缓。当前 ASR 是系统能力实验模式，OCR 是手动资料流和接口预留。

操作步骤：
- 展示 Settings 能力路线图：已支持 / 实验模式 / 待接入 / 暂缓。

画面重点：
- 明确边界。
- 评委能看到决赛路线。

不要展示：
- 任何“已完成真实 OCR/ASR provider”的误导文案。

## 10. 收尾 20 秒

讲解词：
ClassMate 复赛版已经形成从课堂资料到知识点、证据、微测、复习、练习和导出的闭环。决赛版会优先接入真实 vivo ASR/OCR provider，增强多来源证据融合和学习画像，让它更像一个真正的课堂学习 App。

操作步骤：
- 回到首页或课程详情。
- 停在课程概览或导出报告页面。

画面重点：
- 一句话收束：官方模型路径 + 学习闭环 + 诚实扩展路线。

不要展示：
- 任何密钥或隐私材料。
