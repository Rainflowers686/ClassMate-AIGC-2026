# Stage 8C 2 分钟 Demo 视频脚本

> 适用场景：复赛提交材料中的短视频展示（约 2 分钟）。
> 不讲太多技术细节，强调学习闭环、vivo 能力、端侧真机 proof 和可打印报告。
> 节奏：快速、流畅、展示核心差异。

---

## 视频分段（总计约 120 秒）

| 段落 | 时长 | 主题 |
|------|------|------|
| 1 | 10s | 问题引入 |
| 2 | 20s | 学习闭环展示 |
| 3 | 20s | vivo 云端 + 端侧架构 |
| 4 | 15s | 端侧真机 proof |
| 5 | 20s | 图片/拍照 → 草稿 → 分析 |
| 6 | 15s | Ask / Quiz / Review |
| 7 | 10s | 可打印报告 |
| 8 | 10s | 结尾 |

---

## 段落 1：问题引入（0:00–0:10）

**画面：** 快速切入——课堂笔记散乱的特写 → 过渡到 ClassMate App 首页。

**旁白：**
"课后复习，笔记不完整、不知道重点、不知道自己学会了没有。ClassMate 解决这三个问题。"

**画面重点：** 问题 → 解决方案的快速切换。

---

## 段落 2：学习闭环（0:10–0:30）

**画面：** 从首页 → 导入课堂文本 → Timeline 知识点列表（快速滚动）→ 点击一个知识点展示 Evidence 原文引用。

**旁白：**
"ClassMate 不是又一个 AI 总结工具。它是学习闭环：课堂资料变身结构化知识点，每个知识点都有原文证据，不是模型瞎编的。你看到的不只是答案，还有答案从哪来。"

**画面重点：** Timeline 卡片 → Evidence quote。

---

## 段落 3：vivo 云端 + 端侧能力（0:30–0:50）

**画面：** 快速切到 Settings → Official BlueLM 云端配置 → 端侧 BlueLM 3B 诊断页（init/generate OK）→ 来源标签 CLOUD_BLUELM / LOCAL_FALLBACK。

**旁白：**
"vivo 云端 BlueLM 是主引擎，高品质课程分析和问答。端侧 BlueLM 3B 在手机本地运行——断网也能分析、数据不离开手机。vivo 官方 SDK、真实对接、真机跑通。"

**画面重点：** 云端和端侧双状态展示。

---

## 段落 4：端侧真机 Proof（0:50–1:05）

**画面：** 快速展示：文件管理器 `/sdcard/1225` 模型目录 → Settings 诊断 text init/generate success → callVit code 0 → 多模态 generate success。

**旁白：**
"vivo X300 Pro 真机验证。端侧文本生成、多模态图片理解——四盏绿灯。不是 mock，不是模拟，是真实 SDK 在 vivo 设备上运行。"

**画面重点：** 四个绿色诊断状态。

---

## 段落 5：图片拍照 → 草稿 → 分析（1:05–1:25）

**画面：** 拍照界面 → 拍照 → "正在生成草稿"加载 → 草稿编辑页（展示可编辑的 AI 文字描述）→ 点击"确认并分析"→ Timeline 结果。

**旁白：**
"拍板书、拍课件——端侧多模态理解图片，生成可编辑草稿。你确认了才进入分析，不自动入库、不污染知识库。AI 帮你读懂图，你决定什么是最终内容。"

**画面重点：** 草稿可编辑 + 确认按钮 + Timeline 结果。

---

## 段落 6：Ask / Quiz / Review（1:25–1:40）

**画面：** Ask 问答（答案+证据引用+grounded 标签）→ Quiz 微测选择题 → Review "需要多练"列表 → Practice 搜索词卡片。

**旁白：**
"学了就测。Ask 基于课堂问答、Quiz 自动出题、Review 追踪薄弱点。Practice 给你搜索方向——不是爬取，是引导你去找合适的练习资源。"

**画面重点：** 快速切换四个模块的关键画面。

---

## 段落 7：可打印学习报告（1:40–1:50）

**画面：** Export Center → PDF 预览（知识点摘要 → 证据引用 → 微测结果 → 复习建议）→ 展示排版效果。

**旁白：**
"学习成果导出为可打印报告。知识点、证据、微测结果、复习计划——一份完整的课后学习档案。"

**画面重点：** PDF 结构完整 + 打印友好排版。

---

## 段落 8：结尾（1:50–2:00）

**画面：** ClassMate logo + "学习闭环，vivo 智能"副标题 + 四大关键词浮现：学习闭环 / 端侧真机 / 证据可追溯 / 可打印报告。

**旁白：**
"ClassMate —— vivo 蓝心驱动的学习闭环。不只是 AI 总结，是真正帮你学会。"

**画面重点：** 品牌收束 + 核心信息强化。

---

## 注意事项

1. **不讲技术细节**：不出现"callVit"、"validator"、"JSON parse"等术语。
2. **不讲未完成能力**：不暗示 ASR/OCR/TTS 等已完成。
3. **不展示密钥/路径/账号**：所有画面脱敏。
4. **不攻击竞品**：不讲"比XX更强"。
5. **端侧诚实**：不暗示端侧质量等于云端。
6. **不展示 DeepSeek/Compatible**：只展示 BlueLM 官方路径。
