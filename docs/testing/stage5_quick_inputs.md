## 音频转写稿测试文本：大学物理电磁感应

今天这节课我们讲电磁感应。先从磁通量说起，磁通量可以理解为穿过某一面积的磁场总量，它和磁感应强度、面积以及磁场与面积法线之间的夹角有关。公式上常写成磁通量等于 B、S 和夹角余弦的乘积。请注意，磁通量变化才是产生感应电动势的关键，不是只要有磁场就一定会有感应电流。

接下来是法拉第电磁感应定律。它说明闭合回路中的感应电动势大小，等于穿过回路磁通量变化率的负值。负号不是装饰，而是和楞次定律对应，表示感应电流的方向总是阻碍引起它的磁通量变化。比如磁铁靠近线圈时，线圈会产生一个磁场来阻碍磁通量增加；磁铁远离时，线圈又会产生相反方向的磁场来阻碍磁通量减少。判断方向时，不要只背结论，要先判断原磁通量是增加还是减少，再用阻碍变化的思路确定感应磁场方向，最后用右手定则判断电流方向。

第三个重点是动生电动势。当导体棒在磁场中切割磁感线运动时，导体内部自由电荷受到洛伦兹力作用，正负电荷分离，从而在导体两端形成电势差。常见表达式是感应电动势等于 B、l、v 的乘积，但这个公式只适用于导体长度、速度和磁场方向两两垂直的简单情形。如果方向不垂直，就要取垂直分量。

最后讲自感、互感和能量转换。自感是同一线圈中电流变化引起自身磁通量变化，再产生感应电动势的现象；互感是一个线圈电流变化使另一个线圈中产生感应电动势。电磁感应不是凭空产生能量，发电机中机械能转化为电能，变压器中能量通过磁场耦合传递。复习时要把磁通量、法拉第定律、楞次定律、动生电动势、自感和能量转换联系起来，而不是孤立背公式。

## 视频字幕测试文本：C++ 指针与引用

[00:00:05] 这一节我们讲 C++ 中的指针和引用。先看地址，任何变量在内存中都有一个存放位置，取地址运算符可以得到变量的地址。指针变量保存的不是普通数值，而是某个对象的地址。

[00:00:32] 声明指针时要写出它指向对象的类型，比如 int 指针表示它保存的是 int 对象的地址。通过解引用运算符，可以访问指针所指向的对象。解引用前必须确认指针有效，否则可能访问非法内存。

[00:01:05] 空指针表示当前不指向任何对象，现代 C++ 推荐使用 nullptr。空指针可以作为一种安全的初始状态，但不能直接解引用。很多初学者把指针没有初始化和空指针混为一谈，这是错误的。

[00:01:35] 引用可以看作对象的别名。引用必须在定义时初始化，并且初始化之后不能再绑定到另一个对象。引用本身不像指针那样保存可变地址，因此语义上更适合表达“这个参数一定对应某个已有对象”。

[00:02:08] 函数参数传递有三种常见形式：值传递、指针传递和引用传递。值传递会复制实参，函数内部修改副本不会影响原对象。指针传递可以通过地址修改原对象，但要处理空指针风险。引用传递语法更接近普通变量，常用于需要修改实参的场景。

[00:02:45] const 引用很重要。当函数只需要读取对象而不修改对象时，可以使用 const 引用。这样既避免了大对象复制，又保证函数不会修改实参。比如传递字符串、向量或自定义对象时，const 引用通常比值传递更高效。

[00:03:18] 总结一下，指针强调地址和可空状态，引用强调别名和必须绑定。解引用需要安全检查，空指针不能使用，引用必须初始化，函数参数传递要根据是否修改对象、是否允许为空、是否避免复制来选择。

## PPT OCR 测试文本：数项级数判别法

标题：数项级数收敛判别法

1. 数项级数：a1 + a2 + ... + an + ...
2. 部分和 Sn = a1 + a2 + ... + an
3. 若 Sn 有有限极限，则级数收敛；否则发散

必要条件：
若级数收敛，则 an -> 0
注意：an -> 0 不是充分条件
例：调和级数 1 + 1/2 + 1/3 + ... 发散

常用判别法：
- 几何级数：|q| < 1 收敛，|q| >= 1 发散
- p 级数：1 / n^p，当 p > 1 收敛，当 p <= 1 发散
- 比较判别法：用已知级数比较正项级数
- 比值判别法：lim |a(n+1)/an| = L
  L < 1 收敛，L > 1 发散，L = 1 失效
- 根值判别法：lim n 次根号 |an| = L
  L < 1 收敛，L > 1 发散，L = 1 失效

注意事项：
先检查通项是否趋于 0。
正项级数优先考虑比较、比值、根值。
交错级数要检查单调趋零。
绝对收敛一定收敛，条件收敛不绝对收敛。

## 板书 OCR 测试文本：离散数学偏序关系

偏序关系 R：定义在集合 A 上
要求：
1 自反：任意 a 属于 A，有 aRa
2 反对称：若 aRb 且 bRa，则 a = b
3 传递：若 aRb 且 bRc，则 aRc

例：集合 {1,2,3,6} 上的整除关系
1 | 2，1 | 3，1 | 6
2 | 6，3 | 6
每个元素都整除自身
所以整除关系是偏序关系

哈斯图：
省略自反边
省略由传递性推出的边
只画覆盖关系
1 在下方，6 在上方
2 和 3 不可比较

极大元：没有比它更大的元素
最大元：大于等于所有元素
最小元：小于等于所有元素
极小元：没有比它更小的元素
注意：最大元若存在一定唯一；极大元可以有多个。
