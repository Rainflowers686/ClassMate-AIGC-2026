## 初步归因

选择最可能的 1-2 个：

- [ ] **路径问题** — 模型文件路径不存在或不可访问
- [ ] **权限问题** — 缺少存储或读取权限
- [ ] **Native 问题** — `.so` 加载失败、ABI 不匹配
- [ ] **模型问题** — 模型文件损坏或版本不匹配
- [ ] **代码问题** — bridge reflection 调用错误、参数错误
- [ ] **UI 问题** — 状态显示文案错误或缺失
- [ ] **配置问题** — LlmConfig 参数设置错误
- [ ] **NPU/APU 问题** — 芯片不可用或驱动不兼容
- [ ] **未知** — 需进一步排查
