## 1. 高数文本课堂：数项级数判别法

今天我们继续学习数项级数。数项级数可以看成无限多个数相加，关键不是把所有项真的加完，而是考察部分和数列有没有极限。若部分和 Sn 收敛到一个有限值，就说级数收敛；如果部分和没有有限极限，就说级数发散。第一个必要条件是通项趋于零：如果级数收敛，那么 an 必须趋于零。注意这只是必要条件，不是充分条件，调和级数的通项趋于零，但它仍然发散。

几何级数是最基础的例子，形如 a 加 aq 加 aq 的平方一直下去。当公比绝对值小于一时收敛，当公比绝对值大于等于一时发散。p 级数也很重要，形如一除以 n 的 p 次方。当 p 大于一时收敛，当 p 小于等于一时发散。很多比较判别法都要拿几何级数或 p 级数做参照。

对于正项级数，比较判别法要求我们找到一个已知收敛或发散的级数。如果 0 小于等于 an 小于等于 bn，且 bn 收敛，那么 an 收敛；如果 an 大于等于 bn 大于等于 0，且 bn 发散，那么 an 发散。比值判别法看相邻两项绝对值之比的极限 L，L 小于一收敛，L 大于一发散，L 等于一时判别失效。根值判别法看 n 次根号下 an 绝对值的极限，也有同样的结论。

交错级数还要看符号交替、绝对值单调递减并趋于零。绝对收敛一定收敛；如果级数本身收敛但绝对值级数发散，就叫条件收敛。做题时先检查通项是否趋于零，再判断是正项、交错还是一般项，最后选择比较、比值或根值等方法。

## 2. 物理音频转写稿：电磁感应

今天讲电磁感应。先看磁通量，磁通量表示穿过某个面积的磁场总量，它和磁感应强度、面积以及磁场方向与面积法线的夹角有关。磁通量变化是产生感应电动势的关键，不是有磁场就一定有感应电流。

法拉第电磁感应定律说，闭合回路中的感应电动势大小等于穿过回路磁通量变化率的负值。这个负号对应楞次定律。楞次定律强调感应电流的方向总是阻碍引起它的磁通量变化。比如磁铁靠近线圈，原磁通量增加，线圈产生的感应磁场要阻碍这个增加；磁铁远离线圈，原磁通量减少，线圈产生的感应磁场要阻碍这个减少。

动生电动势来自导体在磁场中切割磁感线。导体棒运动时，内部自由电荷受到洛伦兹力，正负电荷分离，在两端形成电势差。简单情况下，动生电动势等于 B、l、v 的乘积，但这要求磁场、长度和速度方向互相垂直。方向不垂直时要取垂直分量。

自感是一个线圈中电流变化引起自身磁通量变化，从而产生感应电动势。互感是一个线圈的电流变化使另一个线圈产生感应电动势。发电机体现机械能转化为电能，变压器体现能量通过磁场耦合传递。复习时要把磁通量、法拉第定律、楞次定律、动生电动势、自感和能量转换联系起来。

## 5. PPT OCR 文本：级数判别法

标题：数项级数判别法

定义：
级数 = a1 + a2 + ... + an + ...
部分和 Sn = a1 + a2 + ... + an
Sn 有有限极限，则级数收敛

必要条件：
级数收敛 => an -> 0
注意：an -> 0 不是充分条件
调和级数通项趋零但发散

常用判别：
几何级数 |q| < 1 收敛
p 级数 p > 1 收敛，p <= 1 发散
比较判别法：与已知级数比较
比值判别法：lim |a(n+1)/an| = L
L < 1 收敛，L > 1 发散，L = 1 失效

## 6. 板书 OCR 文本：偏序关系

偏序关系 R：
集合 A 上的关系

三个性质：
自反：任意 a 属于 A，有 aRa
反对称：aRb 且 bRa，则 a = b
传递：aRb 且 bRc，则 aRc

例：{1,2,3,6} 上整除关系
1 | 2，1 | 3，1 | 6
2 | 6，3 | 6
2 与 3 不可比较

哈斯图：
省略自反边
省略传递边
只画覆盖关系

极大元：没有更大的元素
最大元：大于等于所有元素
最大元若存在一定唯一
