# Ask This Lesson 问题库

用途：真机测试 Ask This Lesson 是否能基于“本节课材料”回答问题、引用证据，并在课堂材料没有依据时拒绝胡编。本文只包含测试问题和预期行为，不包含任何真实密钥、认证信息、完整模型输入、供应商原始返回或内部思考文本。

状态说明：

- `grounded`：课堂文本中有明确依据，答案应引用对应证据。
- `partial`：课堂文本只支持部分回答，答案应说明边界。
- `not_found`：课堂文本没有依据，答案应明确说本节课资料中未找到依据。
