## 样例 1：高数课件截图

### 图片中应包含的文字

```

内容要点：
- 幂级数的定义：Σ a_n (x - c)^n
```

- 选择题：`Σ (x^n / n^2)` 的收敛域是？A. (-1,1) B. [-1,1] C. (-1,1] D. [-1,1)

### Report 中呈现

- 来源标注："高数课件截图·端侧理解"

## 样例 4：C++ 代码截图

### 图片中应包含的文字

```cpp
// 指针与引用示例
#include <iostream>
using namespace std;

void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    cout << "Before: x=" << x << ", y=" << y << endl;
    swap(x, y);
    cout << "After: x=" << x << ", y=" << y << endl;
    return 0;
}
```

### 期望图像理解输出

- 识别代码主题：指针与引用/交换函数
- 识别函数签名：`swap(int& a, int& b)`
- 识别关键概念：引用传递、值交换
- 描述代码的功能（不逐行翻译）

### 不应编造的内容

- 不要编造代码的输出结果
- 不要精确还原全部代码（端侧 VIT 对代码的精确识别有限）
- 不要添加代码中没有的注释

### Ask 测试问题

- "这段代码演示了什么概念？"
- "swap 函数的参数是什么类型？"

### Quiz 测试问题

- 选择题：`swap(int& a, int& b)` 中的 `&` 表示？A.取地址 B.引用 C.指针 D.位与
- 判断题：这段代码使用值传递交换变量。（T/F）

### Report 中呈现

- 来源标注："代码截图·端侧理解"
- 内容摘要："指针与引用示例：swap 函数使用引用传递"
- 标注"代码为端侧理解，建议核对原始代码"

## 样例 6：错误/模糊图片样例

### 图片描述

1. **严重模糊图片**：手指遮挡镜头、运动模糊、低光照
2. **不完全包含内容的图片**：只拍到课件左上角、拍到一半的板书
3. **非学习内容图片**：风景照、自拍、纯色背景

### 期望图像理解输出

- 模糊图片：输出"图片质量不足以进行分析，请重新拍摄"
- 不完整图片：输出"图片内容不完整，请确保拍摄到全部内容"
- 非学习图片：输出"图片中未检测到学习相关内容"

### 不应编造的内容

- 对于模糊/不完整/非学习图片，**不编造任何内容**
- 不应给出"看起来像是..."的猜测

### Ask 测试问题

- 对模糊图片提问：应提示"无法分析此图片，请重新上传"
- 对风景照提问：应提示"未检测到学习内容"

### Quiz 测试问题

- 对非学习图片：不应生成题目

### Report 中呈现

- 不呈现无内容的图像理解条目
