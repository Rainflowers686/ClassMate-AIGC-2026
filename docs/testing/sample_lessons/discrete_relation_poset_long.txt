同学们，今天我们讲离散数学中的关系与偏序。先看二元关系的定义。给定集合 A 和集合 B，A 到 B 的一个二元关系 R 是笛卡尔积 A×B 的一个子集。也就是说，如果有序对 (a,b) 属于 R，就表示 a 和 b 满足这个关系。当 A=B 时，我们说 R 是集合 A 上的二元关系。关系可以用有序对集合表示，也可以用关系矩阵表示，还可以用关系图表示。关系矩阵适合做计算，关系图适合观察结构。

集合 A 上的关系常见性质有自反性、对称性、反对称性和传递性。自反性是指对任意 a 属于 A，都有 (a,a) 属于 R。对称性是指如果 (a,b) 属于 R，那么 (b,a) 也属于 R。反对称性不是“完全没有对称”，而是如果 (a,b) 和 (b,a) 同时属于 R，就只能推出 a=b。传递性是指如果 (a,b) 属于 R，并且 (b,c) 属于 R，那么 (a,c) 也属于 R。判断这些性质时必须检查全体元素，不能只看几个例子。

等价关系要求同时满足自反、对称和传递。一个等价关系会把集合划分成若干互不相交的等价类。比如整数集合上“模 3 同余”是等价关系，因为每个整数与自己同余；如果 a 与 b 同余，那么 b 与 a 同余；如果 a 与 b 同余，b 与 c 同余，那么 a 与 c 同余。这个关系把整数分成三个等价类：余数为 0 的类、余数为 1 的类、余数为 2 的类。同一个等价类内部元素彼此等价，不同等价类之间没有元素等价。

偏序关系要求满足自反、反对称和传递。常见例子包括集合之间的包含关系和整数之间的整除关系。偏序关系不要求任意两个元素都能比较，因此它不同于全序。为了展示偏序结构，我们常画哈斯图。哈斯图会省略自反边，也会省略由传递性推出的边，只保留覆盖关系，并通常把较大的元素画在较高位置。读哈斯图时，如果从 x 可以沿向上的路径到 y，就表示 x 小于等于 y。

在偏序集中，还要区分极大元、最大元、极小元和最小元。极大元是指没有比它更大的可比较元素；最大元则要求它大于等于集合中的所有元素。最大元如果存在一定唯一，但极大元可以有多个。极小元和最小元的关系类似。常见错误是把“没有元素比它大”误认为“它比所有元素都大”。当集合中存在不可比较元素时，极大元不一定是最大元。掌握这些概念后，我们就能分析关系性质、等价类划分和偏序结构。
