# Stage 6 Markdown 导入测试样例

用途：验证 Markdown 导入、纯文本提取、知识点生成、证据定位和异常表现。每个样例都可直接复制到 Markdown 文件或导入文本框。

## 1. UTF-8 中文 Markdown

### 可复制内容

```markdown
# 高等数学：数项级数

今天学习数项级数。级数收敛的定义是：部分和数列存在有限极限。

## 必要条件

如果级数收敛，那么通项必须趋于零。注意，通项趋于零不是充分条件。

## 例子

调和级数的通项趋于零，但调和级数发散。
```

### 预期导入后纯文本效果

保留标题和正文语义，去掉 Markdown 标记后仍能读出“数项级数、部分和、通项趋零、调和级数”。

### 预期知识点

- 数项级数收敛定义
- 通项趋零必要条件
- 调和级数反例

### 失败时表现

若标题或正文丢失，Timeline 可能只有泛泛总结；若 Markdown 标记未清理，证据 quote 可能带多余符号。

## 2. UTF-8 BOM Markdown

### 可复制内容

```markdown
﻿# 大学物理：电磁感应

磁通量表示穿过某面积的磁场总量。磁通量变化会产生感应电动势。

法拉第电磁感应定律说明，感应电动势与磁通量变化率有关。

楞次定律说明，感应电流方向总是阻碍引起它的磁通量变化。
```

### 预期导入后纯文本效果

BOM 不应变成可见乱码；标题应显示为“大学物理：电磁感应”。

### 预期知识点

- 磁通量
- 法拉第电磁感应定律
- 楞次定律

### 失败时表现

若 BOM 未处理，标题前可能出现奇怪字符，课程库聚合和报告标题可能异常。

## 3. 表格 Markdown

### 可复制内容

```markdown
# 判别法对比

| 判别法 | 适用对象 | 结论 |
|---|---|---|
| 比较判别法 | 正项级数 | 与已知级数比较 |
| 比值判别法 | 含阶乘或指数项 | 极限小于一收敛 |
| 根值判别法 | n 次幂结构 | 极限小于一收敛 |

比值判别法在极限等于一时失效。
```

### 预期导入后纯文本效果

表格应被转成可读的行文本，不应丢失“适用对象”和“结论”。

### 预期知识点

- 比较判别法
- 比值判别法
- 根值判别法
- 判别法失效条件

### 失败时表现

若表格被整段丢弃，生成的知识点会偏少；若竖线过多，证据 quote 可能不自然。

## 4. 数学公式 Markdown

### 可复制内容

```markdown
# p 级数

p 级数写作：

$$
\sum_{n=1}^{\infty}\frac{1}{n^p}
$$

当 p > 1 时级数收敛；当 p <= 1 时级数发散。

公式只是表达形式，做题时还要判断是否能和 p 级数比较。
```

### 预期导入后纯文本效果

公式可保留为文本形式，关键结论“p > 1 收敛，p <= 1 发散”不能丢。

### 预期知识点

- p 级数形式
- p 级数收敛条件
- p 级数比较用途

### 失败时表现

若公式清洗过度，证据中可能只剩结论；若清洗不足，公式符号可能影响分段。

## 5. 链接和图片 Markdown

### 可复制内容

```markdown
# 离散数学：偏序关系

偏序关系需要满足自反、反对称和传递。

[课程参考链接](https://example.invalid/course)

![哈斯图示意](local-hasse-demo.png)

画哈斯图时省略自反边和由传递性推出的边，只保留覆盖关系。
```

### 预期导入后纯文本效果

链接文本可保留，URL 可保留或简化；图片不能被当成已解析图像，图片 alt 文本可作为普通文本参考。

### 预期知识点

- 偏序关系三性质
- 哈斯图画法
- 图片未解析边界

### 失败时表现

若系统声称已解析图片，属于误导；若 URL 影响知识点提取，应调整清洗策略。

## 6. 代码块 Markdown

### 可复制内容

```markdown
# C++ 指针与引用

指针保存对象地址，引用是对象别名。

```cpp
int x = 10;
int* p = &x;
int& r = x;
```

引用必须初始化，空指针不能直接解引用。
```

### 预期导入后纯文本效果

代码块可以保留为普通文本，但知识点应来自“指针保存地址、引用是别名、引用必须初始化、空指针不能解引用”。

### 预期知识点

- 指针保存地址
- 引用是别名
- 引用必须初始化
- 空指针不能解引用

### 失败时表现

若代码块被当成自然语言解释，可能生成过细的语法题；若代码块完全删除，证据可能缺少示例。

## 7. 混合高数课堂 Markdown

### 可复制内容

```markdown
# 高等数学课堂：级数综合复习

今天把数项级数的几类判别法串起来。首先，级数收敛要求部分和数列有有限极限。其次，若通项不趋于零，级数一定发散。

## 正项级数

正项级数常用比较判别法、比值判别法和根值判别法。比较判别法要找到已知收敛或发散的参照级数。

## 交错级数

交错级数判别法要求绝对值单调递减并趋于零。若绝对值级数也收敛，则称为绝对收敛；若原级数收敛而绝对值级数发散，则称为条件收敛。

## 易错点

1. 通项趋零不能推出级数收敛。
2. 比值判别法 L = 1 时不能下结论。
3. 不要把条件收敛误判为绝对收敛。
```

### 预期导入后纯文本效果

标题、分节、编号列表都能转成自然课堂文本。适合生成 6-8 个知识点。

### 预期知识点

- 级数收敛定义
- 通项趋零必要条件
- 正项级数判别法
- 交错级数判别法
- 绝对收敛
- 条件收敛
- 常见易错点

### 失败时表现

若只生成 1-2 个知识点，说明 Markdown 清洗或 compact draft 提示需要调优；若证据 quote 找不到，检查清洗前后文本是否一致。
