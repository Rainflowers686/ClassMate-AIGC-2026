## 1. 高数课件 OCR：级数判别法

**场景说明**

老师展示“数项级数收敛判别法”PPT，页面是知识点列表和判别流程图。

**可粘贴 OCR 文本**

```text
数项级数收敛判别法
1. 级数：a1 + a2 + ... + an + ...
2. 部分和：Sn = a1 + a2 + ... + an
3. 若 Sn 有有限极限 S，则级数收敛；否则发散
4. 收敛必要条件：an -> 0
5. 注意：an -> 0 不是充分条件，调和级数发散
6. 几何级数：|r| < 1 收敛，|r| >= 1 发散
7. p 级数：p > 1 收敛，p <= 1 发散
8. 正项级数优先考虑比较判别法、极限比较判别法
```

**场景说明**

**可粘贴 OCR 文本**

```text
例：判断 Σ n! / 3^n 的敛散性
设 an = n! / 3^n
a(n+1) / an = [(n+1)! / 3^(n+1)] * [3^n / n!]
= (n+1) / 3
lim a(n+1)/an = +∞
因为 L > 1，所以级数发散
比值判别法：
L = lim |a(n+1)/an|
L < 1 绝对收敛
L > 1 发散
L = 1 方法失效
```

**可能的 OCR 错字版本**

```text

an=n!/3^n
a(n十1)/an = (n十1)/3
lim a(n+1)/an = +∞
因为 L>l，所以级数发散
L=l 方法失效
```

**期望知识点**

- 比值判别法适用场景
- 相邻项比值
- L>1 发散
- L=1 方法失效
- 阶乘型通项

**期望证据引用**

## 6. C++ 课件 OCR：指针与引用

**场景说明**

C++ 课件页对比指针、引用和函数参数传递。

**可粘贴 OCR 文本**

```text
C++ 指针与引用
指针：保存对象地址，可以为空，可以重新指向
解引用：通过指针访问所指对象
空指针：不指向有效对象，使用前必须检查
野指针：未初始化或值不可预测的指针
悬空指针：指向生命周期已经结束的对象

引用：对象的别名，声明时必须初始化
引用通常不能重新绑定到另一个对象
值传递：复制实参
指针传递：复制地址，可通过解引用修改原对象
引用传递：形参是实参别名，可直接修改原对象
const 引用：避免复制，保证不修改实参
```
