## 1. 高数文本课程输入

今天我们讲数项级数的收敛判别法。一个数项级数就是把一列数按顺序相加，写成 a1 加 a2 加 a3 一直到无穷。判断它是否收敛，首先看必要条件：如果级数收敛，那么通项 an 必须趋于 0。注意这只是必要条件，不是充分条件；调和级数的通项趋于 0，但级数仍然发散。第二类基本例子是几何级数，公比绝对值小于 1 时收敛，公比绝对值大于等于 1 时发散。第三类是 p 级数，1 除以 n 的 p 次方，当 p 大于 1 时收敛，当 p 小于等于 1 时发散。比较判别法适合处理正项级数，如果一个级数的项被一个已知收敛级数控制，则它也收敛；如果它控制了一个已知发散级数，则它发散。比值判别法关注相邻两项比值的极限，极限小于 1 收敛，大于 1 发散，等于 1 时不能判断。根值判别法关注 n 次根的极限，也有同样的判断边界。最后讲交错级数，如果项的绝对值单调趋于 0，交错级数可以收敛；但还要区分绝对收敛和条件收敛。绝对收敛比条件收敛更强，绝对收敛一定收敛，条件收敛只是原级数收敛而绝对值级数发散。

## 2. 物理课堂文本输入

这节课讲电磁感应。磁通量表示磁场穿过某个面积的多少，常写成磁感应强度与面积以及夹角余弦的乘积。当穿过闭合回路的磁通量发生变化时，回路中会产生感应电动势，这就是法拉第电磁感应定律。它告诉我们，感应电动势的大小与磁通量变化率成正比。楞次定律说明感应电流的方向总是阻碍引起它的磁通量变化，因此它体现了能量守恒。比如磁铁靠近线圈时，线圈产生的磁场会阻碍磁铁靠近；磁铁远离时，线圈产生的磁场会阻碍它远离。动生电动势是导体在磁场中运动切割磁感线产生的电动势，常见公式是 B、l、v 的乘积，但要注意方向由右手定则判断。自感是一个线圈中电流变化引起自身磁通量变化，从而在自身产生感应电动势。互感是一个线圈电流变化在另一个线圈中产生感应电动势。电磁感应的本质不是凭空产生能量，而是机械能、电能和磁场能之间的转换。

## 3. 离散数学图论输入

今天我们用图论语言复习关系结构。一个有向图可以表示集合上的二元关系，顶点表示集合元素，有向边表示元素之间满足关系。无向图适合表示对称关系，因为边没有方向。路径是从一个顶点沿边走到另一个顶点的序列，回路是起点和终点相同的路径。连通性描述图中顶点之间是否能够通过路径到达。在有向图中，要区分强连通和弱连通：强连通要求任意两个顶点可以互相到达，弱连通只要求忽略方向后连通。树是一类没有回路的连通无向图，n 个顶点的树有 n 减 1 条边。生成树是在连通图中包含所有顶点的一棵树。图的度数表示与顶点相关联的边数，有向图还要区分入度和出度。欧拉路径关注是否能把每条边走一次，哈密顿路径关注是否能把每个顶点访问一次。图论模型常用于课程先修关系、知识点依赖和状态转移。

## 4. C++ 代码课堂输入

今天讲 C++ 指针、引用和函数参数传递。指针变量保存的是对象地址，取地址符可以得到变量地址，解引用可以访问地址指向的对象。空指针表示当前不指向任何有效对象，使用前必须判断，否则可能导致程序崩溃。引用可以理解为对象的别名，引用必须在定义时初始化，初始化后不能再绑定到另一个对象。函数参数传递有三种常见方式：值传递会复制实参，函数内部修改不影响外部变量；指针传递传入地址，可以通过解引用修改外部对象，但要考虑空指针；引用传递语法更像普通变量，也能修改外部对象。const 引用适合传入大对象且不希望被修改，可以避免复制开销。数组名在很多表达式中会退化为指向首元素的指针，但数组不是普通指针。理解地址、生命周期和所有权，是避免悬空指针、野指针和内存泄漏的基础。

哈斯图画法：
