# Stage 8B 端侧文本审核 SDK 接入规划

> 官方已提供端侧文本审核 SDK（`CmsLocalFrame`），本文规划其接入 ClassMate 的方案。
> 不声称 SDK 已集成，不运行 Gradle，不修改生产代码。

---

## 1. 官方 SDK 接口

```
CmsLocalFrame.init(context, listener) → 初始化
TextModeration(text, callback, timeout) → 审核文本

ResponseResult.data.result:
  0 → 通过（内容安全）
  1 → 嫌疑（需人工复核）
  2 → 不通过（内容不合规）

Listener:
  onInitSuccess()
  onInitFailed(code, message)
```

---

## 2. 接入点设计

### 2.1 StudyReport 导出前

- **时机**：用户点击"导出报告" → 报告生成完成 → 审核 → 导出
- **审核内容**：AI 生成的报告摘要、学习建议、薄弱点分析
- **result=1**：标注"此段内容需人工复核"，用户可选择是否保留
- **result=2**：自动移除该段落，替换为"[此段内容经审核已移除]"
- **审核失败**：跳过审核，标注"本地审核暂不可用，内容未经审核"

### 2.2 分享前

- **时机**：用户点击"分享" → 审核分享内容 → 生成分享卡片
- **审核内容**：分享的文本内容（学习摘要、成绩、排名）
- **result=1**：弹出确认框"内容可能包含敏感信息，确认分享？"
- **result=2**：阻止分享，提示"内容不合规，无法分享"
- **审核失败**：弹出确认框"审核服务暂不可用，确认分享？"

### 2.3 AI 生成报告润色后

- **时机**：AI 润色报告完成 → 审核 → 展示给用户
- **审核内容**：润色后的全文
- **result=1**：高亮嫌疑段落
- **result=2**：退回润色前版本，标注原因
- **审核失败**：直接展示润色结果（标注"未经审核"）

### 2.4 AI 练习解释后

- **时机**：AI 生成练习题目解析 → 审核 → 展示
- **审核内容**：题目解析文本
- **result=1**：在解析末尾添加"[此解析需教师复核]"
- **result=2**：替换为"此题目解析暂不可用"
- **审核失败**：展示解析（标注"未经审核"）

### 2.5 多模态图片理解文本入库前

- **时机**：端侧 VIT 生成理解文本 → 审核 → 加入 MaterialBundle
- **审核内容**：端侧图像理解生成的文本
- **result=1**：标记 confidence=LOW，加入入库但标注"需复核"
- **result=2**：丢弃理解文本，提示用户"图片理解结果未通过审核"
- **审核失败**：使用 confidence 兜底（端侧理解本身已标注实验性）

### 2.6 用户自定义模型输出入库前

- **时机**：用户使用自定义模型生成的文本 → 审核 → 入库
- **审核内容**：自定义模型输出
- **result=1/2**：同 2.5 逻辑
- **审核失败**：标注"未经审核"后入库

## 3. 安全原则

| 原则 | 说明 |
|------|------|
| 不替代云端安全审查 | 端侧审核是**补充**，云端审核仍是主力 |
| 不上传内容 | 端侧审核在本地完成，审核的文本不离开设备 |
| 审核失败不阻塞 | 审核 SDK 异常时，降级为"未经审核"标注，不阻塞用户流程 |
| 不导出审核结果 | 审核 result 代码不写入导出报告（避免反向利用） |
| 不记录审核内容 | 日志仅记录审核耗时和 result 代码，不记录被审核的文本 |

## 4. UI 文案

### result=0（通过）

```
无需特别 UI 提示，正常流程继续。
```

### result=1（嫌疑）

```
⚠️ 内容审核提示
此段内容已通过本地审核，但系统建议您复核。
[查看高亮段落] [确认保留] [移除内容]
```

### result=2（不通过）

```
⚠️ 内容未通过审核
此段内容不符合内容安全规范，已被系统移除。
[了解原因]
```

### 审核服务不可用

```
ℹ️ 本地审核暂不可用
当前内容未经本地审核，请谨慎处理。
```

## 5. Fallback 策略

| 场景 | 行为 |
|------|------|
| `CmsLocalFrame.init` 失败 | 禁用本地审核，所有内容标注"未经审核" |
| `TextModeration` 超时 | 跳过本次审核，标注"审核超时" |
| `ResponseResult` 解析失败 | 视为审核服务异常，标注"审核异常" |
| SDK AAR 不存在 | 编译期不依赖 `CmsLocalFrame`，运行时降级 |
| 审核结果异常（非 0/1/2） | 视为审核异常 |
