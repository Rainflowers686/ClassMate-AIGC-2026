# Product vision

## Who

University students in concept-dense courses — 高数, 计算机, 大学物理 — who struggle to organise
knowledge, find weak spots, and build a review plan *after* class.

## The wedge

Not another note app, not another AI summariser. ClassMate's difference:

1. The model genuinely **understands** the lecture (real concepts, merged duplicates).
2. Every result is **traceable to original-text evidence**.
3. Micro-tests serve **learning** (understanding / judgment / application / error-analysis /
   transfer), not "which line matches the text".
4. The **review plan adapts** to answers and feedback.
5. Built for competition demo from day one, with room to grow for finals.

## Core loop

input text → BlueLM understands → real knowledge points → each bound to evidence → valuable
micro-tests → answer → review plan → feedback optimises the next round.

## What "good" looks like (级数 lecture)

Real knowledge points such as: 级数的定义 · 部分和数列 · 级数收敛与发散 · 等比级数 · p 级数 ·
正项级数判别法 · 交错级数 · 绝对收敛与条件收敛 — each with original-text evidence and a
learning-oriented question.

## Non-goals (round 1)

- No audio capture / ASR (text input only).
- No account system or cloud sync.
- Local fallback is a safety net, never the headline intelligence.
