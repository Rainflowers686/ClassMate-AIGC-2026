# “需要多练”搜索词库

用途：为 Review / Weakness Hub 的“需要多练”动作生成外部搜索关键词。这里只生成搜索词或搜索入口，不爬取平台、不下载内容、不解析第三方网页。

安全与合规原则：

- 只打开外部搜索，不做第三方平台爬取。
- 导出报告中只保留搜索词和来源说明，不保留抓取结果。
- 用户自行判断资源版权、质量和适用性。
- 视频或题库入口只是补充资源，不替代课堂原文证据。
