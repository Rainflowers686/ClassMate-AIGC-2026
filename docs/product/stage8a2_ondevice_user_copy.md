# Stage 8A-2 端侧模型用户界面文案

> 全部中文，适合 Settings UI / 关于页面 / 模型状态面板使用。
> 不声称端侧真机已跑通,仅描述功能边界和用户可见行为。

---

## 一、端侧 BlueLM 3B 说明

### 标题
端侧智能助手（BlueLM 3B）

### 副标题
您的设备本地运行的轻量级语言模型

### 简介
端侧智能助手使用 BlueLM 3B 参数量模型在您的设备本地运行推理，无需联网即可完成简单的问答和文本处理任务。模型运行在设备本地，您的对话内容不会离开设备。

### 功能说明
- 支持本地文本对话（需要下载模型文件）
- 支持端侧多模态实验（图像理解，需要设备支持）
- 当端侧模型不可用时，自动回退到云端模型或本地规则
- 初始化过程需要约 10–30 秒（因设备性能而异）

---

## 二、端侧多模态实验

### 标题
端侧多模态（实验性）

### 副标题
在设备本地理解图片内容

### 简介
端侧多模态功能允许您在设备本地对图片进行理解和分析。图片数据仅在设备本地编码处理，不会上传至任何云端服务器。

### 说明
- 此功能为实验性质，识别准确率可能低于云端模型
- 图片通过设备的视觉编码器（VIT）本地处理
- 编码后的特征向量在本地模型中使用，原始图片不会保存
- 需要设备具备足够的计算能力

## 三、模型路径

### 标题
端侧模型存储路径

### 说明
端侧模型文件存储在以下位置（应用私有目录）：
```
/sdcard/Android/data/<package>/files/models/bluelm3b/
```
- 模型文件在首次启用端侧功能时自动下载
- 下载需要 Wi-Fi 连接（默认设置）
- 模型文件大小约 2–4 GB，请确保有足够存储空间
- 卸载应用时会一并删除模型文件

## 八、隐私与数据说明

### 端侧数据不上传
- 端侧对话的文本内容**不会上传**至任何服务器
- 端侧处理的图片**不会上传**至任何服务器
- VIT 编码在设备本地完成，仅特征向量参与推理

### 原始 Prompt 不保存
- 应用**不保存**您输入的任何原始问题文本
- 对话记录仅在当前会话内存中保留
- 关闭应用或结束会话后，对话上下文将被清除

### 权限说明
- **存储权限**：用于下载和存放端侧模型文件
- **相机权限**（可选）：用于拍摄图片进行端侧多模态分析
- **相册权限**（可选）：用于选择图片进行端侧多模态分析
- **网络权限**：用于下载模型文件和云端模型对话
- 端侧功能**不需要**定位、通讯录、短信等其他权限

## 九、设置入口文案

### 端侧模型开关
```
[开关] 启用端侧智能助手
在设备本地运行 BlueLM 3B 模型。开启后，简单问答优先使用本地模型处理。
```

### 端侧模型下载
```
[按钮] 下载端侧模型（约 2–4 GB）
下载完成后，端侧智能助手即可离线使用。建议在 Wi-Fi 环境下进行。
```

### 多模态实验开关
```
[开关] 启用端侧多模态（实验性）
在设备本地对图片进行理解分析。开启后，图片处理优先使用本地视觉编码器。
```

### 模型偏好
```
[选择] 模型偏好
• 自动（推荐）：优先端侧，不可用时使用云端
• 仅端侧：仅使用本地模型，不进行网络请求
• 仅云端：仅使用云端模型（需要网络连接）
```
